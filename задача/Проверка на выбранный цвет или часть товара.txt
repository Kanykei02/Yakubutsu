Проверка на выбранный цвет или часть товара

+1) Определить клик
+2) Добавить по умолчанию стали, бесцветный border
+3) Проверить существования класса
  3.1) Если нет класса с помощь jq добавить стиль css border
4) После проверки - если есть класс то вывести окно успешного добавления товара в корзину
5) После проверки - если нет класс то вывести красный бордер там где не выбрано
6) Не добавлять товар пока не выбранны параметры
